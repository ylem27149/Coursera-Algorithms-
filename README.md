## Week 1: Percolation - UnionFind

一个区域，里面的格子有打开/关闭两种状态，水会从上方流进相邻的打开格子。
判断格子是否打开（isOpen)，格子里是否有水（isFull），水能否流入区域的最下方（percolates)。
